# 정보 산업의 발전

## 1. 정보 산업의 이해

### 정보 산업의 개념과 특징

> - 정보 사회란 컴퓨터와 정보 통신 기술의 발전에 기초해 이루어진 사회이다
> - 인간의 주요 활동이 정보 및 통신 기술이 제공하는 서비스의 지원을 받아 이루어지는 사회이다
>
> #### 정보 산업의 범위
>
> 하드웨어 산업 + 소프트웨어 산업 + 네트워크 산업 + 콘텐츠 산업
>
> #### 정보 산업의 특징
>
> 1. 고도의 기술 집약형 첨단 산업
> 2. 자원 절약형 산업
> 3. 무공해 산업
> 4. 다른 산업 시스템 분야에 광범위하게 이용
> 5. 전 세계 정보 혁명에 의한 대표적인 성장 산업이며 전략 산업이다

### 정보 기술의 역할

>1. 지식 집약 방식의 지식 정보 사회로 새로운 패러다임 형성
>2. 인터넷, 위키, 소셜 네트워크, 모바일 등 개인 미디어의 보급으로 정보의 자유로운 소통 촉진과 민주주의의 잠재력 신장
>3. 정보 기술과 관련된 새로운 직업 발생 유도
>4. 가상적으로 사회관계를 맺을 수 있는 인터넷 공동체 형성
>5. 문화 생산이 자유로운 각종 도구의 발전으로 문화적 다양성 증대
>6. 사람들의 욕구를 충족시키고 사회적 쟁점 해결
>7. 기후 변화와 환경 오염, 테러, 및 안보 위협 증가, 에너지 및 자원 고갈 등의 문제 해결

## 정보 산업의 발전 과정과 전망

### 정보 산업의 발전 과정

> - 하드웨어 산업 비중이 높음(1980) -> 소프트웨어 산업 비중이 높아짐(1990)
> - 정보 처리 기술과 통신 기술은 각각 독립적으로 박전하다 어느 시점부터 하나로 융합하여 발전
>
> 컴퓨터 등장 -> 기관 위주 컴퓨팅 시대 -> 개인 컴퓨팅 시대 -> 사람 간의 컴퓨팅 시대 -> 유비쿼터스 컴퓨팅 시대

### 정보 산업의 전망

>정보 기술이 국가 사회 시스템의 변화와 혁신을 선도하는 역할
>
>> -> 여러 나라에서 정보 기술과 사회 잘전을 함께 이끄는 정보화 전략 추진
>
